# 小红书收藏自动分类插件

一个 Chrome 浏览器扩展，自动识别小红书笔记内容并智能分类到对应专辑。

## 功能特点

- **智能分类**：根据笔记标题、描述、标签自动匹配预定义分类
- **全自动操作**：点击收藏后自动完成分类流程，无需手动干预
- **自定义分类**：支持添加、删除、修改分类及关键词（每个分类 50-100+ 个关键词）
- **本地存储**：分类配置保存在浏览器本地，数据安全

## 安装步骤

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，输入 `chrome://extensions/`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用方法

1. 打开小红书网页版（https://www.xiaohongshu.com）并登录
2. 进入任意笔记详情页
3. 点击笔记下方的星星图标 ⭐️ 收藏
4. 插件自动完成：识别内容 → 匹配分类 → 选择/创建专辑

## 设置分类

1. 点击浏览器工具栏中的插件图标
2. 点击"设置分类"按钮
3. 在设置页面中修改、添加、删除分类
4. 点击"保存设置"

## 默认分类（15个）

美食、旅行、穿搭、学习、美妆、家居、健身、数码、影视、职场、生活、理财、宠物、母婴、汽车
